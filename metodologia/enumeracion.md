# Metodologia para vulnerar una máquina.

## 1. Reconocimiento

- Recopilar información inicial
- Identificar sistema operativo

## 2. Escaneo de puertos

- Detectar puertos abiertos
- Identificar servicios
- Revisar versiones

## 3. Enumeración

### Web

- Analizar tecnologías
- Buscar rutas
- Revisar configuraciones

### Servicios

- Analizar cada servicio encontrado
- Buscar usuarios
- Buscar vulnerabilidades

## 4. Explotación

1. Encontrar vulnerabilidad
2. Obtener acceso inicial
3. Mantener sesión

## 5. Privilege Escalation

- Revisar permisos
- Buscar configuraciones débiles
- Conseguir privilegios elevados

## 6. Post-explotación y documentación
- Confirmar nivel de acceso conseguido
- Recopilar evidencias
- Registrar cada paso
- Guardar pruebas
- Explicar vulnerabilidad, impacto y solución
